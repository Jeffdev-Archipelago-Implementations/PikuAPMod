# Pikuniku Archipelago

Archipelago.gg implementation for Pikuniku.

## Installation

Extract the `Pikuniku_Archipelago` folder into the `BepInEx/plugins` folder in your game folder.

Requires BepInEx 5 to be installed for Pikuniku.

## Usage

Launch the game and press F1 to open the Archipelago connection menu. Enter your
hostname, port, and slot name, then hit Connect.
